Введение
На протяжении нескольких лет ученической жизни я пользуюсь пропуском для прохода в школу, покупки обеда или оплаты еды в буфете. Но нередко я забывал пропуск дома, или попросту он терялся. Проанализировав данную проблему, я смог найти множество решений данного вопроса. Об одном из таких решений, я хочу поделиться.

Актуальность
	В России миллионы школьников пользуются пропусками ежедневно. Все школьники когда-либо забывали или теряли пропуск. По этой причине я решил написать программу для замены или дополнения системы пропусков, а именно модель для распознавания лица. С помощью моей программы можно будет облегчить жизнь школьникам, а также обезопасить школы от неизвестных личностей. 
Использованные программные решения
•	Python – высокоуровневый язык программирования общего назначения с динамической строгой типизацией и автоматическим управлением памятью
•	HTML – стандартизированный язык гипертекстовой разметки документов для просмотра веб-страниц в браузере
•	Pycharm – среда разработки
•	OpenCV – библиотека алгоритмов компьютерного зрения, обработки изображений и численных алгоритмов общего назначения с открытым кодом
•	Face Recognition – фреймворк для распознавания лица
•	Eel – фреймворк для разработки web-приложений
•	Bootstrap – свободный набор инструментов для создания сайтов и веб-приложений
Обзор проектного решения
После анализа проекта, мной было принято решение использовать язык программирование python из-за его возможности выбрать множество направлений разработки. В ходе длительного предпроектного исследования я обнаружил множество библиотек, которые можно было бы использовать. Но выбрал именно OpenCV за их простоту и удобность использования.
